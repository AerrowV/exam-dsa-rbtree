## Beskrivelse
    Red-black tree er en selv-balancerende binært søgetræ, som overholder følgende fem regler:

        1. Hver node skal enten være rød eller sort.
        2. Rod-noden er altid sort. 
        3. Alle NIL-noder (null referencer) betragtes som sorte.
        4. En rød node parent må aldrig være rød (ingen to røde noder i træk).
        5. For enhver node gælder: alle simple stier fra noden til dens NIL-nodes skal indholde det samme antal sorte noder.
